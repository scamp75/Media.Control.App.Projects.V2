wpf 테스트박스를 이용하여 타임코드 00:00:00:00 형태의 00 단위로 변경 하는 usercontrol 를 작성 한다.
1. 00:00:00:00 형태로를 항상 유지 하는데 : 는 삭제 되지 않는다.
2. :: 사이에 마우스를 클릭 하면 00 두자리를 자동 선택 된다.
3. 자동 석택한 자리에 숫자만 입력 되며, HH:mm:ss:ff 에 시,분, 초 , 프래임 에 값이 맞지 않은 경우 00으로 초기화 한다. ex) 분 67를 입력하면 00으로 변경
4. delete 키를 누르면 값을 삭제가능하다.
5. enter 키를 누르면 완료 시킨다.
